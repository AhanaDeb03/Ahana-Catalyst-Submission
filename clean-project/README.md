# TalentAI Scout 🎯

**Precision AI Talent Scouting Agent for High-Growth Teams**

TalentAI Scout is an end-to-end recruitment platform that leverages advanced AI to automate candidate sourcing, technical screening, and initial outreach. Built for the modern recruiter, it turns hours of manual screening into minutes of high-value decision-making.

## ✨ Core Features

### 1. **AI-Powered Precision Scoring**
Paste any Job Description and watch as our AI Agent cross-references it against candidate profiles. It doesn't just look for keywords; it analyzes:
- **Technical Depth**: Alignment with your specific tech stack.
- **Strategic Fit**: Industry experience (e.g., FinTech scale) and career trajectory.
- **Actionable Insights**: Instant "Match Reasons" and "Gaps" for every profile.

### 2. **Fully Automated Pipeline (Zero-Click)**
The entire recruitment funnel is automated from the moment you paste a Job Description.
- **Instant Background Evaluation**: As soon as scouting is complete, the AI Agent automatically begins technical interviews with all qualified candidates simultaneously.
- **Dynamic Interest Scoring**: Watch your dashboard update in real-time as the AI conducts chats, calculates interest scores, and summarizes candidate motivations.
- **Deep-Dive Monitoring**: Click "View Analysis" on any candidate to jump into their live AI-led interview and see the raw data behind the scores.

### 3. **Premium CRM Dashboard**
A sophisticated, glassmorphism-inspired interface designed for clarity and speed.
- **Visual Rankings**: Priority-sorted shortlist with combined Match + Interest scores.
- **Pipeline Overview**: Monitor your entire recruitment funnel at a glance.

## 🌐 Deployment Instructions

This project is ready to be deployed to **Vercel** or **Netlify** with zero configuration.

### **Option A: Vercel (Recommended)**
1. Push this code to a GitHub repository.
2. Go to [vercel.com](https://vercel.com) and click **"New Project"**.
3. Import your repository.
4. Click **"Deploy"**. Vercel will automatically detect the Vite configuration.
